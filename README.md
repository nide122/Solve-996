# Solve-996
为了解决程序员996之痛，解决开发过程中大量重复性代码。实现编程自动化，目前已实现Java编程自动化。做软件开发行业的RPA。
# 技术实现思路介绍
将开发过程中大量重复性、规则性的代码，生成模版以及规则，开发人员只要输入模型，即可自动生成这部分代码，目前Java开发已实现40%的开发效率提升。
